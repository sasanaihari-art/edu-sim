# 子ども教育資金シミュレーター

大学進学などの教育費を、**高校卒業（18歳の3月）までにいくら貯まるか**試算し、
目標達成に必要な追加積立額を自動計算するWebツールです。

## ▶ ライブ版（そのまま使えます）

**https://sasanaihari-art.github.io/edu-sim/**

ブラウザで開くだけ。インストール不要です。

## できること

- お子さんごとに目標額・保有銘柄・口座区分（NISA/特定）を登録
- 「現状 → 高校卒業時の到達見込み（税引後）」を表示
- 目標達成に必要な毎月の積立額を自動計算
- 「もし月◯円積み立てたら」のシミュレーション
- 下振れ・主軸・上振れのグラフ表示
- eMAXIS Slim オルカン / S&P500 の基準価額をボタンで自動取得

## 🤖 自分用にカスタマイズしたい人へ

このページのコードを **AI（クロコ＝Claude Code／チャピコ＝Codex など）に読ませて**、
「**私用に作り直して**」と頼めば、あなた仕様のシミュレーターが作れます。

例：扱うファンドを変える・項目を増やす・デザインを変える など、ご自由にどうぞ。

## 🔒 プライバシー

- 入力したデータは **あなたの端末内（ブラウザの localStorage）にのみ保存** されます
- **サーバーには一切送信されません**（通信するのは基準価額の取得時のみ）
- 閲覧者ごとにデータは独立しているので、他人にあなたの入力が見えることはありません

## ⚠️ ご注意

- 計算は前提条件にもとづく**試算**です。将来の運用成果を保証するものではありません
- 投資判断はご自身の責任でお願いします
- 基準価額の自動取得は eMAXIS Slim オルカン / S&P500 の2本のみ対応。
  それ以外のファンドは基準価額を手動で入力してください
